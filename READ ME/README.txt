- To get the full content of the website: 

 * Extract all the zip folder.

 * If you are using xampp, make sure to save the whole project floder inside 
   C://xampp/htdocs folder

 * You're going to create a database with the name of (quiz).
   And a database with the name of (content).

 * Then you will have to import the quiz.sql file into quiz database you've created.
   And import content.sql file into content database you've created.

 * Go to your browser, then type in the search bar:
   http://localhost/yourIELTS/index.html
   to go to home page and explore the whole website.
